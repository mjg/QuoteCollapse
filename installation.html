Installation

To install, download the xpi below (right-click, save as) from your browser. Then, in Thunderbird, use Tools->Extensions->Install to install the xpi you just downloaded; restart Thunderbird.

  • quotecollapse-1.1.xpi: Version 1.1 for Mozilla Thunderbird 50.0 - 60.*, SeaMonkey 2.0.*-2.7.* improves upon the partial collapsing support
  • quotecollapse-1.0.xpi: Version 1.0 for Mozilla Thunderbird 50.0 - 53.*, SeaMonkey 2.0.*-2.7.* adds partial collapsing
  • quotecollapse-0.9.xpi: Version 0.9 for Mozilla Thunderbird 3.0b1 - 10.*, SeaMonkey 2.0.*-2.7.* adds support for SeaMonkey explicitly
  • quotecollapse-0.8.xpi: Version 0.8 for Mozilla Thunderbird 3.0b1 - 3.1.*, SeaMonkey 2.0.* (tested with various nightlies, 3.0pre, 3.1pre) adds support for Thunderbird 3.0
  • quotecollapse-0.7.xpi: Version 0.7 for Mozilla Thunderbird 0.7 - 1.5 (tested with 1.5RC1) and Mozilla 1.7 - 1.8b adds support for Thunderbird 1.5
  • quotecollapse-0.6.xpi: Version 0.6 for Mozilla Thunderbird 0.7 - 1.5 (tested with 1.5b1) and Mozilla 1.7 - 1.8 adds support for Thunderbird 1.5b
  • quotecollapse-0.5.xpi: Version 0.5 for Mozilla Thunderbird 0.7 - 1.0 (tested with 1.0) and Mozilla 1.7 - 1.8 adds theme support and a smaller hotspot
  • quotecollapse-0.4.xpi: Version 0.4 for Mozilla Thunderbird 0.7 - 1.0 (tested with 0.7.3, 0.8, 0.9, 1.0rc) and Mozilla 1.7 - 1.8 (tested with 1.8a5)
  • quotecollapse-0.3.xpi: Version 0.3 for Mozilla Thunderbird 0.7 - 0.9 (tested with 0.8, 0.9) and Mozilla 1.7 - 1.8 (tested with 1.8a4) adds an icon and fixes a typo in contents.rdf
  • quotecollapse-0.2.xpi: Version 0.2 for Mozilla Thunderbird 0.7 - 0.8+ (tested with 0.7, 0.8) adds additional functionality with SHIFT, CTRL and CTRL+SHIFT modifiers.
  • quotecollapse-0.1.xpi: Version 0.1 for Mozilla Thunderbird 0.7 - 0.8+ (tested with 0.8)
