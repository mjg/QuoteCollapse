Source Code

The quotecollapse source code can be accessed in either of the following ways:

  • View the source using the CVS web interface
  • Check out the source using anonymous CVS with the following commands (password is guest): 
    cvs -d :pserver:guest@mozdev.org:/cvs login
cvs -z3 -d :pserver:guest@mozdev.org:/cvs co quotecollapse
  • View the source using the Git web interface
  • Check out the source using Git with either of the following commands: 
    git clone git://github.com/mjg/QuoteCollapse.git
git clone http://github.com/mjgQuoteCollapse.git
